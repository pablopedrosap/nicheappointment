Set Up Gmail Access

To automate your email campaigns, we need access to your Gmail account. Follow these steps to create an App Password:

  1. Go to your Google Account settings.
  2. Navigate to the Security section.
  3. Under "Signing in to Google," select "App Passwords."
  4. Generate a new App Password for "Mail" and "Other (Custom name)."
  5. Copy the generated password and paste it below.
